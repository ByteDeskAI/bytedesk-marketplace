# fleet-judge-sidecar

Newline-delimited JSON sidecar that proxies the fleet web server's intelligence calls (`judge_state`, `drift`, `estimate_cost`) to Claude Haiku via [`@anthropic-ai/claude-agent-sdk`](https://www.npmjs.com/package/@anthropic-ai/claude-agent-sdk). The Go server (`fleet/web/server/judge_haiku.go`) launches this process at startup with `node index.mjs` when `ANTHROPIC_API_KEY` is present, writes one JSON request per line to its stdin, and reads one JSON response per line from its stdout. On sidecar death, missing key, or per-request timeout the Go side falls back to the `heuristicJudge` provider, so the dashboard remains correct without it. Run `npm install` once inside this directory before first use; the model defaults to `claude-haiku-4-5-20251001` and can be overridden with the `MODEL` env var.
